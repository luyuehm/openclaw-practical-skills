# openclaw-practical-skills

A curated, practical, open-source skill pack for OpenClaw agents.

## Included skills

- `self-healing`: incident recovery workflow (reproduce → diagnose → minimal fix → evidence-based verification → rollout → learning capture)
- `release-readiness`: pre-release verification checklist and execution workflow

## Suggested usage

1. Copy `skills/*` into your agent workspace skill directory.
2. Run with your project context and toolchain.
3. Keep each skill small, auditable, and evidence-driven.

## Philosophy

- Fix small, verify big.
- Prefer reproducible commands over verbal confidence.
- Convert repeated manual checks into scripts/checklists.
